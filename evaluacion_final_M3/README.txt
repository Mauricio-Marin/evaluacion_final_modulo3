DESCRIPCION DEL SISTEMA

Este sistema de información está realizado para la mejora de gestión, control,
 seguridad y disponibilidad de información de una empresa de asesorías de prevención
 de riesgos.
 
 Contiene módulos de gestión de información de los clientes y de la empresa
 que provee el servicio, a través de un conjunto de páginas web.
 

 
----------------------------------------------------------------------------------

INSTRUCCIONES DE INSTALACION Y USO

Debe copiar la carpeta en su disco y desde allí abrir los archivos .HTML en un navegador.
Cada página WEB posee un menú para dirigirse a la página que necesite. 
Las páginas son los siguientes:

Index.html: página de inicio del sistema

Módulo usuarios:
	-login.html: permite acceder al listado de usuarios del sistema
	-listadodeusuarios.html: muestra el listado de usuarios, pudiendo ser editados o eliminados.
	-crearusuario.html: formulario para crear un nuevo usuario.
	
Módulo capacitaciones:
	-listarcapacitaciones: muestra el listado de capacitaciones.
	-crearcapacitacion.html: formulario para crear capacitaciones.
	-administrarasistentes: muestra una lista de asistentes y un formulario para agregar un asistente
	a una capacitación.
	
Módulo asesorías:
	-listadoasesorias.html: muestra un listado de asesorías y permite ver las actividades de mejora
	de cada asesoría.
	-crearasesorias.html: formulario para crear nuevas asesorías.
	-listadovisitas.html: muestra un listado de visitas y permite validar los chequeos realizados a cada visita.
	-listadoaccidentes.html: formulario que permite ingresar un nuevo registro de accidente.
	
Modulo pagos:
	-listadopago.html: muestra la lista de pagos.
	-crearpago.html: formulario para crear un nuevo pago en la lista de pagos.
	
reporte.html: muestra un informe con gráficos sobre los clientes por comuna, accidentes vs visitas
y cantidad de chequeos.
